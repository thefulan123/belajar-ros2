=== Dasar ROS2 / modul/ ===

Modul pembelajaran teori ROS2 dalam format markdown.
Urutan sesuai official ROS2 Humble Beginner Tutorials.

Daftar:
   01-environment.md           → Sourcing ROS2 & environment variable
   02-turtlesim.md             → Pengenalan ROS2 CLI (turtlesim)
   03-node.md                  → Node ROS2
   04-topic-pub-sub.md         → Topic, Publisher, Subscriber
   05-service.md               → Service ROS2
   06-parameter.md             → Parameter ROS2
   07-action.md                → Action ROS2
   08-workspace-package.md     → Workspace, Package, Build, Sourcing
   09-launch.md                → Launch file
   10-callback-timer.md        → Callback, event, timer
    11-micro-ros.md             → micro-ROS di Arduino
    12-custom-interfaces.md     → Custom .msg / .srv / .action

Baca secara berurutan dari 01 ke 12.
Setiap modul punya contoh kode di folder explore/ yang sesuai.
