=== Dasar ROS2 / launch/ ===

Folder untuk launch file ROS2 (.launch.py atau .xml).

Launch file digunakan untuk menjalankan banyak node
sekaligus dengan satu perintah.

Cara pakai:
  ros2 launch <package_name> <file.launch.py>

Atau langsung dari folder ini:
  ros2 launch -p Dasar ROS2/launch/nama_file.launch.py
