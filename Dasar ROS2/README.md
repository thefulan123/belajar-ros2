# Dasar ROS2 — Level 1

**ID:** Level pertama pembelajaran ROS2 untuk pemula. Pelajari 12 modul teori + praktik langsung di explore.

**EN:** First level of ROS2 learning for beginners. Study 12 theory modules + hands-on practice in explore.

---

## Struktur / Structure

| Folder | Isi / Content |
|--------|---------------|
| `modul/` | 12 modul teori ROS2 (01–12) |
| `explore/` | 7 praktik standalone Python/Arduino |
| `src/` | Workspace colcon (Track 2 — ROS2 packages) |
| `docs/` | Dokumentasi pendukung (ERRORS, KORELASI, disclaimer) |
| `scripts/` | Helper scripts (source_ros.sh) |
| `launch/` | Launch files |
| `config/` | YAML config parameters |
| `tests/` | Test files |
| `logs/` | Agent action log |

---

## Cara Belajar / How to Learn

1. Baca modul berurutan dari 01 ke 12
2. Setiap modul → langsung praktik di folder explore/ terkait
3. Setelah semua modul → coba Track 2 di `src/` untuk integrasi penuh
4. Lanjut ke [Penerapan ROS2 pada Komponen](../Penerapan%20ROS2%20pada%20Komponen%20yang%20Umum%20ditemui/) untuk hardware

Lihat [KORELASI.md](docs/KORELASI.md) untuk mapping modul ↔ komponen hardware.
